NSCLC TNM Staging

AJCC 9th edition stage grouping for non-small cell lung cancer. Enter T, N, and M categories to see the corresponding clinical stage.

T Category

T1a / T1b / T1c / T2a / T2b / T3 / T4

N Category

N0 / N1 / N2 / N3

M Category

M0 / M1a / M1b / M1c

Stage Group

Select T, N, and M categories then press Calculate Stage.

This tool is intended for educational reference only and follows simplified AJCC 9th edition NSCLC stage grouping. Always confirm final staging with multidisciplinary review and institutional guidelines.
